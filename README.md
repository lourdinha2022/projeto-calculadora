# 📌 Projeto Calculadora (Shell Script + Python)

## 📖 Descrição
Este projeto consiste na criação de uma calculadora simples utilizando Shell Script e Python, executada via terminal no ambiente Linux.

O usuário informa dois números e escolhe a operação matemática, e o sistema retorna o resultado.

---

## ⚙️ Tecnologias utilizadas
- Shell Script (.sh)
- Python 3
- Linux

---

## ▶️ Como executar o arquivo .sh

1. Abra o terminal
2. Navegue até a pasta do projeto
3. Dê permissão de execução:

```bash
chmod 744 calculadora.sh
./calculadora.sh
python3 --version
python3 calculadora.py
